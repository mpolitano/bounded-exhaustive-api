## Build BEAPI from source code

### Requirements

- Java 1.8 (`BEAPI` might not work on newer versions)

### Compile **BEAPI**

Run the following gradle command to generate **BEAPI**'s .jar:

```./gradlew singleJar```

Compiled jar is located in: `./libs/randoop-all-3.0.6.jar`
